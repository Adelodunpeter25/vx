# VX Editor

A modern vi text editor written in Go.

## Features

- **Syntax highlighting** for 200+ languages
- Search with `/` and navigate with `n/N`
- Undo/redo with `u` and `r`
- Clipboard support

### Installation

```bash
go build -o vx cmd/vx/main.go
```
